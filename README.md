# CSS Gradient Generator

A visual CSS gradient generator with live preview. Pick colors, adjust stops, choose gradient type (linear, radial, conic), set angle/position, and copy the CSS.

**ğŸ”— Live:** [gradient.colmena.dev](https://gradient.colmena.dev)

## Features

- Linear, radial, and conic gradient types
- Add/remove color stops with color picker and position slider
- Angle slider for linear & conic gradients
- Click-to-place position picker for radial gradients
- Large live preview area
- One-click CSS copy
- 6 preset gradients to start from
- Clean dark UI

## Run Locally

```bash
bun install
bun dev
```

## Build

```bash
bun run build
```

## Stack

TypeScript, React, Vite, Tailwind CSS
